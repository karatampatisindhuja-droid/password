### Password Generator – Theory (15 Lines)

1. This program generates random passwords using Python.
2. It uses the `random` module to create unpredictable passwords.
3. The user enters the number of passwords to generate.
4. The user also specifies the length of each password.
5. A minimum password length of 3 characters is maintained.
6. The `generatePassword()` function creates passwords using lowercase letters.
7. Random letters are selected from the alphabet string.
8. The generated letters are combined to form a password.
9. The `replaceWithNumber()` function replaces some letters with digits.
10. This increases the security of the password.
11. The `replaceWithUppercaseLetter()` function converts some letters to uppercase.
12. Uppercase letters make the password stronger and harder to guess.
13. All generated passwords are stored in a list.
14. The program displays each password with its corresponding number.
15. This project demonstrates the use of functions, loops, strings, lists, and randomization in Python.
